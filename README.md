# Rock-_-Paper_-Scissors
Built a multi-round Rock-Paper-Scissors console game in C++ under the guidance of Eng. Mohamed Abou Hadhood (Programming Advices).
The game allows the user to play up to 9 rounds against the computer with full control over choices, round results, and an overall game summary.

✨ Key Learning Outcomes:

🧩 Structured the game logic using enums, structs, and functions.

🎲 Implemented random computer choices and validated user inputs.

🎨 Added color-coded feedback for win/lose outcomes.

🔄 Practiced decision-making using nested if-else and switch statements.

🔧 Strengthened understanding of loops, conditionals, and modular programming in C++.


🚀 This project was part of my early learning journey into software development and object-oriented thinking.
